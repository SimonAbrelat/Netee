# Netee

## Setup Instructions
1. Install a modern Linux distro (tested on Ubuntu 20.0.4)
2. Run `sh setup.sh`

## Running
Navigate to the build directory with `cd build`.
### Server
`./game -server -port <port>`
The port option is not required.

### Client
`./game -host <ip> -port <port>`
The port option is not required.
